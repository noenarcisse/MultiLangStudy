# Loops (only the spiciest)
## Go
### for
  for c'est THE loop chez Go. Ca fait tout<br>
  ```go
//la vraie
	for i := 0 ; i < 10 ; i++ {
		fmt.Println(counter)
	}
  ```
  ```go
//le forever
	for {
		fmt.Println("Vers l'infini et au dela")
	}
  ```
  ```go
//le pseudo while
  counter := 0
	for counter < 10 {
		counter++
		fmt.Println(counter)
	}
  ```
